# 🔐 AuthMaster - Full Stack Authentication System

A complete authentication system built with **React** and **Node.js** featuring JWT tokens and a file-based database using fs-extra. No MongoDB required!

## ✨ Features

- 🔐 **User Registration** - Create new accounts with validation
- 🔑 **User Login** - Secure authentication with JWT
- 📁 **File-Based Database** - Uses fs-extra to store users in JSON
- 🛡️ **Protected Routes** - Secure dashboard access
- 💅 **Modern UI** - Clean, responsive design with inline CSS
- ⚡ **Password Hashing** - bcryptjs for secure password storage
- 🎨 **Beautiful Pages** - Login, Register, and Dashboard with animations

## 🚀 Tech Stack

### Frontend
- React 18
- React Router DOM
- Axios
- React Icons

### Backend
- Node.js
- Express
- JSON Web Tokens (JWT)
- bcryptjs
- fs-extra
- uuid
